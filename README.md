# Atv5-sw
Uso de multiplas threads para calcular a soma de cada linha de uma matriz quadrada

## Istallation
```bash
$ git clone 
$ cd Atv5-sw
```

## Commands
```bash
$ make        # Compila o arquivo c       
$ make run    # Executa o arquivo c
$ make clean  # Limpa o binario gerado do arquivo c
```
